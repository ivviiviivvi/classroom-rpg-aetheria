## üìù Contributing Research

### Research Upload Guidelines

1. **File Naming Convention**
   - Use descriptive, kebab-case names
   - Include date for time-sensitive research: `user-study-2025-12.md`
   - Prefix with category: `ANALYSIS_`, `STUDY_`, `REPORT_`, `BENCHMARK_`

2. **Document Structure**
   - Start with title and metadata (date, author, type)
   - Include executive summary
   - Present methodology and findings
   - Conclude with recommendations

3. **Review Process**
   - Research documents should be peer-reviewed before merging
   - Link to relevant issues or discussions
   - Update this README index when adding new documents

### Research Types

- **Analysis** - Systematic evaluation of product features or architecture
- **Studies** - User testing, surveys, and behavioral research
- **Reports** - Compiled findings from various sources
- **Benchmarks** - Performance and comparison metrics
- **Audits** - Compliance, accessibility, and security reviews
