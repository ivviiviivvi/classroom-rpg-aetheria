## Phase 2: Recording (2-3 hours)

### Voiceover Recording Setup

**Equipment needed** (minimum):
- Decent microphone (USB mic like Blue Yeti or even good earbuds)
- Quiet room with soft surfaces (closet with clothes works great)
- Audio recording software (Audacity - free, or Adobe Audition)

**Recording tips**:
1. Do a 10-second test recording to check levels
2. Stand up while recording for better vocal projection
3. Keep water nearby, clear your throat before starting
4. Record in segments (each Act separately) to make editing easier
5. Do 2-3 takes of each section, pick the best later
6. Leave 2 seconds of silence at start and end for clean editing

**Quality checklist**:
- [ ] No background noise (fans, traffic, computer hum)
- [ ] Consistent volume throughout
- [ ] Clear articulation (no mumbling or rushing)
- [ ] Natural pacing with strategic pauses
- [ ] Energy appropriate to content (not monotone, not over-excited)

### Headshot Recording (if applicable)

**Equipment needed**:
- Webcam or smartphone camera (1080p minimum)
- Ring light or natural window light
- Clean, uncluttered background

**Recording tips**:
1. Position camera at eye level
2. Frame yourself in lower right third of screen (leave space for graphics)
3. Look slightly off-camera toward script/teleprompter
4. Record 2-3 versions of each segment
5. Keep facial expressions subtle and professional

## Phase 3: Visual Production (4-8 hours)

### Recommended Tools

**Option A: Professional** (paid, full control)
- Adobe After Effects for motion graphics
- Adobe Premiere Pro for editing
- Figma for designing static cards and diagrams

**Option B: Accessible** (free/affordable, easier learning curve)
- DaVinci Resolve (free video editor)

### Animation Guidelines

**Timing Reference**:
- Text fade-in: 0.3 seconds
- Card appearance: 0.5 seconds
- Section transitions: 1 second cross-dissolve
- Hold text on screen: Minimum 2 seconds for readability

**Movement**:
- Use "ease-out" easing for most animations (starts fast, slows down)
- Keep movements subtle (10-20% opacity changes, small position shifts)
- Avoid spinning or bouncing unless it serves a specific purpose

## Phase 4: Editing & Assembly (3-5 hours)

### Editing Workflow

1. **Set up your timeline**:
   - Import all voiceover segments
   - Import music track (if using)
   - Create video tracks for graphics and B-roll

2. **Lay down foundation**:
   - Place voiceover on primary audio track
   - Add music on secondary audio track at -22dB
   - Mark timecodes for each Act on timeline

3. **Add visuals in order**:
   - Start with hook (0:00-0:30)
   - Work through each Act sequentially
   - Sync graphics to appear WITH voiceover, not before

4. **Polish pass**:
   - Add transitions between sections
   - Check audio levels (voiceover should be clearly louder than music)
   - Add captions (most editors have auto-caption features now)
   - Color grade if needed (slight contrast boost, saturation adjustment)

5. **Final review**:
   - Watch full video 2-3 times
   - Check for typos in on-screen text
   - Verify all timings match script
   - Test audio on different devices (laptop speakers, phone, headphones)

### Music Integration

**If including music**:
1. Select ambient track (see script recommendations in Section 7)
2. Fade in over 3 seconds at start
3. Keep volume at -20dB to -25dB (much quieter than voice)
4. Consider removing during "Tradeoffs" section (3:15-3:45) for emphasis
5. Fade out over 3 seconds at end

**Free music sources**:
- YouTube Audio Library (free, no attribution needed)
- Artlist.io (subscription, high quality)
- Epidemic Sound (subscription, huge library)
- Uppbeat (free with attribution)

## Phase 5: Captioning & Accessibility (1-2 hours)

### Adding Captions

**Automated options**:
- YouTube auto-captions (upload, let YouTube generate, download SRT)
- Descript (automatic transcription included)
- Rev.com (paid human captions, most accurate)
- Kapwing (free auto-captions)

**Manual refinement**:
1. Review auto-generated captions for accuracy
2. Break lines at natural speech pauses
3. Keep line length under 42 characters
4. Display 1-2 lines at a time maximum
5. Use white text on black semi-transparent box (70% opacity)

**Styling guidelines**:
- Font: Sans-serif (Arial, Helvetica, Inter)
- Size: Large enough to read on mobile (typically 28-36pt)
- Position: Bottom center, never covering on-screen graphics
- Timing: Appear 0.5 seconds before spoken words for reading time

### Accessibility Checklist

- [ ] Captions included for all spoken content
- [ ] On-screen text has 4.5:1 contrast ratio minimum
- [ ] Color is not the only way information is conveyed (use icons + text)
- [ ] No flashing or strobing effects
- [ ] Audio description provided if complex visuals are shown without narration
- [ ] Transcript available as separate document

## Phase 6: Export & Distribution (1 hour)

### Export Settings

**Primary version** (YouTube, Portfolio, Email):
- Format: MP4 (H.264 codec)
- Resolution: 1920x1080 (1080p)
- Frame rate: 30fps
- Bitrate: 8-10 Mbps (high quality)
- Audio: AAC, 192 kbps, stereo

**LinkedIn version**:
- Same as above but consider shorter 90-second cut
- Square format (1:1, 1080x1080) if posting as native LinkedIn video

**Mobile-optimized version**:
- 720p resolution (smaller file size)
- Vertical format (9:16, 1080x1920) for Stories/Reels if needed

### Pre-Upload Checklist

- [ ] Video plays correctly from start to finish
- [ ] Audio is clear and balanced
- [ ] All text is readable (test on mobile device)
- [ ] Captions are accurate and timed properly
- [ ] End card displays for at least 3 seconds
- [ ] File size is reasonable (<500MB for 4-minute video)
- [ ] Exported in multiple formats if needed

### Distribution Strategy

**Week 1: Soft Launch**
1. Upload to portfolio website (unlisted or password-protected)
2. Share with 3-5 trusted peers for feedback
3. Iterate based on feedback if needed

**Week 2: Public Launch**
1. **LinkedIn**: 
   - Post with compelling caption (see example below)
   - Tag relevant people/companies (sparingly)
   - Post mid-week morning (Tuesday-Thursday, 8-10am)

2. **Portfolio website**: 
   - Feature prominently on homepage
   - Include transcript below video
   - Add context paragraph explaining the project

3. **Direct outreach**:
   - Send to specific hiring managers with personalized message
   - Include in job applications where relevant
   - Add to email signature as "Recent Project Video"

4. **YouTube** (optional):
   - Upload with SEO-friendly title and description
   - Add to relevant playlists
   - Include timestamps in description

## Troubleshooting Common Issues

### "My voiceover sounds flat"
**Fix**: Stand up while recording, smile (it changes your tone), imagine you're explaining to one specific person

### "Visuals look amateurish"
**Fix**: Use fewer elements with more white space, stick to 2-3 colors max, increase font sizes, use professional templates

### "Video feels too long"
**Fix**: Cut the 90-second version first (see Section 8 of main script), test if that feels better

### "Timing is off between voice and visuals"
**Fix**: Make visuals appear WITH voiceover or 0.5 seconds after, never before

### "Audio quality is poor"
**Fix**: Record in smaller room with soft materials, use noise reduction in post, keep mic close to mouth (6-8 inches)

### "Can't afford professional tools"
**Fix**: Use free alternatives: DaVinci Resolve (editing), Canva free (graphics), phone camera (recording), Audacity (audio)

### "Don't want to show my face"
**Fix**: Use avatar_mode="none" and focus on strong voiceover with excellent visuals
