## üêõ Reporting Bugs

If you've found a bug, please help us fix it by:

1. **Searching existing issues** to see if it's already been reported
2. **Creating a new issue** using the bug report template
3. **Providing detailed information**:
   - Steps to reproduce the issue
   - Expected vs. actual behavior
   - Environment details (OS, browser, Node version)
   - Screenshots or error messages if applicable

[Create a Bug Report ‚Üí](https://github.com/ivviiviivvi/classroom-rpg-aetheria/issues/new?template=bug_report.md)
